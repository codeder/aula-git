# Aula de Git - Readme personalizado
###Aprendendo a formatar um arquivo do tipo README para ficar xic a apresentação
***

__Etapas de criação do repositório local:__

1.  mkdir git
2.  cd git

__Etapas de criação do repositório remoto:__

1.  Entrei no Github em minha conta e cliquei em Create new > New repository
2.  Repository name: aula-git. Tipo: Public
3.  Create repository

__Clonando repositório remoto no local:__

1.  git clone https://github.com/edersilva/aula-git.git
2.  cd aula-git
2.  touch README.md
3.  git add README.md
4.  git commit -m "Primeiro commit"
5.  git push origin master
